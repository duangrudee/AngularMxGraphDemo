# NgxGraphEditor

This project shows how to integrate mxGraph to Angular 10 project
following this post on stackoverflow.com

https://stackoverflow.com/questions/49922708/how-to-integrate-mxgraph-with-angular-4


## How to use

```
$ cd mxgraph2
$ npm install
$ cd ../ngx-graph-editor
$ npm install
$ npm start
```
